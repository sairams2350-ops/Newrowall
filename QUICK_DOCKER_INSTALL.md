# ⚡ Quick Docker Install for Your System

## Your System: x64 (AMD64) ✅

**Download this**: Standard Docker Desktop for Windows
👉 **https://www.docker.com/products/docker-desktop/**

Click **"Download for Windows"** (the standard version, not ARM)

---

## 3 Steps

1. **Download** from link above
2. **Install** (check "Use WSL 2")
3. **Restart** computer

Then run:
```cmd
cd D:\neurowall
docker-compose up -d
```

**Done!** 🎉

---

**Note**: You have x64 system, so use the standard Windows version (not ARM).
